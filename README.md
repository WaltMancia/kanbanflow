# KanbanFlow

![Kanban Board Preview](file:///C:/Users/walte/.gemini/antigravity-ide/brain/25ad6c59-f2e4-493f-a590-498056eb7a46/kanban_screenshot_1780805018815.png)

A sleek, modern **Kanban board** web application built to showcase a clean, responsive UI with smooth micro‑animations and glassmorphism styling. This project is perfect for adding to a developer portfolio, demonstrating front‑end skills and design sensibility.

---

## ✨ Features

- **Responsive layout** – works beautifully on desktop, tablet, and mobile.
- **Dynamic cards** – drag‑and‑drop between columns with smooth animations.
- **Customizable columns** – add, rename, or delete columns on the fly.
- **Persisted state** – data saved locally in `localStorage` (no backend required).
- **Elegant design** – pastel palette, subtle gradients, glassmorphism effects, and modern typography (Google Fonts – *Inter*).

---

## 🛠️ Tech Stack

- **HTML5** – semantic structure.
- **Vanilla JavaScript (ES6+)** – core logic, drag‑and‑drop.
- **CSS3** – Flexbox/Grid, custom properties, animations.
- **Google Fonts** – *Inter* for premium typographic feel.

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/kanbanflow.git

# Navigate into the project
cd kanbanflow

# Open the app in your default browser
start index.html   # Windows
# or open "index.html" manually.
```

---

## 🚀 Usage

1. **Create columns** – click the "+ Add Column" button.
2. **Add cards** – click the "Add Card" button inside any column.
3. **Drag & drop** – grab a card and move it to another column.
4. **Persist** – your board layout is saved automatically; reload the page to restore.
